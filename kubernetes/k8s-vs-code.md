## How to 

1. Brdiges to Kubernetes 확장 설치 
2. 확장 후 palettes (command+shift+P or F1) 설치하고 
3. 검색에서 `bridge to kubernetes configure` 검색 
4. 이후는 흐름을 따라서 설치하면 된다.
