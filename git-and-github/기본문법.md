## merge, rebase 

{Current Branch} git merge, rebase {target commit, branch}

- main에 있다면, 

```bash
$ git merge feature 
```

main이 feature branch를 병합한다. 

```bash
$ git rebase feature 
```

main이 feature branch를 재정렬한다. 

## Detached Head 

- Head는 깃의 현재 위치다. 
- 만일 branch로 checkout 하면 non-detached, commit으로 checkout 하면 detached
